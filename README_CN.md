MoLive 是一个 iOS 应用程序，专门用于 小米动态照片 和 Live Photo 之间的相互转换。它提供了简洁的用户界面和便捷的转换功能。

[screenshot](image.png)

### ✅ 已实现功能
  - [x] 小米动态照片 转 Live Photo
  - [x] Live Photo 转 小米动态照片  
  - [x] 支持从相册或文件批量导入（最多10张）
